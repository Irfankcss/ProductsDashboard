# Products Dashboard

Small learning project built to practice **React with TypeScript** and experiment with modern tools such as **TailwindCSS**, **shadcn/ui**, and **Meilisearch**.

The application allows basic product management with features like **CRUD operations, search, filtering, and pagination**.

## Tech Stack

- React
- TypeScript
- TailwindCSS
- shadcn/ui
- Meilisearch
- ASP.NET Core Web API

## Features

- Create, edit and delete products
- Search products using Meilisearch
- Filter products by category
- Pagination
- Modern UI built with TailwindCSS and shadcn
